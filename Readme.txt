1 - Installare QGIS dalla pagina https://download.qgis.org 

2 - Il Plugin è stato realizzato utilizzando la versione di QGIS 3.4 e testato sulla versione 3.10, è sufficiente la versione Standalone di QGIS. Non utilizzare versioni precedenti alla 3.4

3 - Per installare il Plugin su Windows cliccare sul file Install_Plugin.bat altrimenti vedere il punto 8.

4 - Per l'installazione manuale, copiare la cartella "LuniPlugin" in C:\Users\UTENTE\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins
    In alternativa, per raggiungere la directory, dalla schermata di QGIS 3 cliccare su Settings -> User Profiles -> Open active profile folder e recarsi in python\plugins

5 - Per attivare il Plugin dalla schermata di QGIS 3 cliccare su Plugins -> Manage And Install Plugins -> Installed -> Lunigiana e spuntare la casella corrispondente.
 
6 - Creare un nuovo progetto QGIS e dalla cartella ShapeFiels trascinare TUTTI i seguenti file: Elevatione.tif, benigeo.shp, fiumi_lunigiana.shp, comuni_lunigiana.shp, funzionigeo.dbf, funzionigeo_ruoli.dbf

7 - Per accedere alle funzionalità del Plugin, utilizzare il Menù in Plugins -> Lunigiana, o le icone presenti in alto a destra sulla Plugins toolbar.

8 - Per l'installazione su macOS e Linux cliccare su Plugins -> Manage And Install Plugins -> Install From ZIP. Selezionare LuniPlugin in formato .zip e cliccare su Install Plugin.
